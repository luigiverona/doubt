doubt

One-command fresh-Arch workstation setup.

Install

curl -fsSL https://doubt.luigiverona.dev/install | bash

The verified self-contained 1.0.5 release checks the workstation, presents one complete plan, asks once, installs required dependencies and applications, configures supported tools, verifies the result, and cleans its temporary work.

Commands

doubt
doubt plan
doubt verify
doubt pkg list
doubt pkg add SOURCE CATEGORY PACKAGE
doubt pkg remove SOURCE PACKAGE
doubt pkg check
doubt --help

Planning and verification are read-only. Package commands manage declaration files only and never install or remove packages.

Boundary

doubt manages its established packages, GitHub, managed SSH, Git, and Codex setup. It does not manage Arch installation, system upgrades, removals, dotfiles, shells, desktop environments, themes, or generic services.
